### `print`

`print` prints the `v` to the console without appendding a newline.

Signature:

```python
def print(v: int | string):
  ...
```

Example:

```python
print("Hello, World!")
print(123)
```

### `sqrt`

`sqrt` returns the square root of the `n`

Signature:

```python
def sqrt(n: int) -> int:
  ...
```

Example:

```python
sqrt(64) == 8
sqrt(16) == 4
```
