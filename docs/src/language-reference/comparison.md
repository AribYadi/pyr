### Less Than

```python
1 < 2 # result is true
```

### Greater Than

```python
1 > 2 # result is false
```

### Less Than or Equal

```python
2 <= 2 # result is true
```

### Greater Than or Equal

```python
2 >= 2 # result is true
```

### Equal

```python
1 == 2 # result is false
"Hello" == "Hello" # result is true
"Hello" == "World" # result is false
```

### Not Equal

```python
1 != 2 # result is true
"Hello" != "Hello" # result is false
"Hello" != "World" # result is true
```
